It's All Text! tests

These are a series of tests for verifying that It's All Text! is working correctly. I'll be adding more tests to these pages as I discover bugs.

When reporting a problem for a specific test, please include:

  • What URL you were testing.
  • The number of the test.
  • What happened as you performed the test. Make sure you go back to previous tests if you started seeing odd, but not invalid, behavior.
  • Include what you thought should have happened if it doesn't match what was explained in the test.
